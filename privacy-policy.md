# プライバシーポリシー - Sidebar Bookmark Manager

最終更新日: 2024年12月7日

## 概要

Sidebar Bookmark Managerは、ブックマークを美しく管理するためのChrome拡張機能です。
このプライバシーポリシーでは、当拡張機能がどのようにデータを扱うかを説明します。

## 収集するデータ

Sidebar Bookmark Managerは以下のデータのみを使用します：

### ブックマークデータ
- ブックマークのタイトル
- ブックマークのURL
- フォルダ構造

### ユーザー設定
- サイドバーの表示設定
- ユーザーのカスタマイズ設定

## データの使用目的

収集したデータは以下の目的でのみ使用されます：

- ブックマークをサイドバーに表示するため
- ブックマークの整理・管理機能を提供するため
- ユーザー設定を保存・復元するため

## データの保存場所

**すべてのデータはお使いのブラウザ内（ローカルストレージ）に保存されます。**

当拡張機能は外部サーバーにデータを送信することは一切ありません。

## データの共有

**Sidebar Bookmark Managerは、収集したデータを第三者と共有することは一切ありません。**

- 外部サーバーへの送信なし
- 広告ネットワークへの送信なし
- 分析サービスへの送信なし

## 使用する権限

Sidebar Bookmark Managerは以下のChrome API権限を使用します：

### bookmarks
- **目的**: ブックマークの読み取り、作成、編集、削除
- **使用方法**: サイドバーにブックマークを表示し、管理機能を提供

### storage
- **目的**: ユーザー設定の保存
- **使用方法**: サイドバーの設定やユーザーのカスタマイズを保存

これらの権限は、拡張機能の基本機能を提供するために必要不可欠です。

## セキュリティ

お客様のデータのセキュリティは私たちの最優先事項です：

- すべてのデータはローカルに保存されます
- 外部への通信は行いません
- コードはChrome Web Storeの審査を通過しています

## お子様のプライバシー

Sidebar Bookmark Managerは13歳未満のお子様を対象としていません。
13歳未満のお子様から意図的に個人情報を収集することはありません。

## プライバシーポリシーの変更

このプライバシーポリシーは、必要に応じて更新される場合があります。
変更があった場合は、このページに新しいプライバシーポリシーを掲載します。

## Privacy Policy - Sidebar Bookmark Manager (English)

Last Updated: December 7, 2024

### Overview

Sidebar Bookmark Manager is a Chrome extension for beautiful bookmark management. This privacy policy explains how the extension handles data.

### Data Collection

Sidebar Bookmark Manager only accesses:
- Bookmark titles and URLs
- Folder structure
- User preferences

### Data Usage

Data is used solely for:
- Displaying bookmarks in the sidebar
- Managing and organizing bookmarks
- Saving user settings

### Data Storage

**All data is stored locally in your browser.**

The extension does not transmit any data to external servers.

### Data Sharing

**Sidebar Bookmark Manager does not share any data with third parties.**

- No external server communication
- No ad networks
- No analytics services

### Permissions

**bookmarks**: To read, create, edit, and delete bookmarks
**storage**: To save user settings

### Security

Your data security is our priority:
- All data stays local
- No external communication
- Code reviewed by Chrome Web Store
